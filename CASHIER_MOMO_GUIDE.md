# 📱 Cashier Guide: Mobile Money Payments

## Quick Reference for Processing MoMo Payments

---

## 🎯 Step-by-Step Process

### Step 1: Start Payment
1. Add items to cart
2. Click **"Checkout"**
3. Select **"Mobile Money"** tab
4. Enter customer's phone number (e.g., 024 123 4567)
5. Click **"Charge Customer"**

### Step 2: Inform Customer
Tell the customer:
> **"You'll receive an SMS with a code in a few seconds. Please reply to that SMS with the code to approve the payment."**

### Step 3: Wait for Approval
- Customer receives SMS (2-5 seconds)
- Customer replies to SMS with the code
- System shows "Payment Successful!" (5-10 seconds after reply)
- Print receipt

---

## 💬 What to Say to Customers

### When Starting Payment:
> "I'm sending a payment request to your phone now."

### After Clicking "Charge Customer":
> "Check your phone for an SMS. Reply to that message with the code you receive."

### If Customer Looks Confused:
> "You'll get a message like 'Enter code 123456 to pay...'. Just reply with that code - 123456."

### If Customer Asks "How?":
> "Open the SMS, type the code, and send it back. Just like replying to a normal text message."

---

## ⏱️ Timing

- **SMS arrives**: 2-5 seconds after clicking "Charge Customer"
- **Customer replies**: Depends on customer
- **Payment confirms**: 5-10 seconds after customer replies
- **Total time**: Usually 30-60 seconds

---

## ❓ Common Questions

### "I didn't receive the SMS"
- Check phone number is correct
- Wait 10-15 seconds (sometimes delayed)
- Check customer's phone has signal
- Try again if needed

### "The code expired"
- Codes expire after 10 minutes
- Start a new payment
- Customer will get a new code

### "I replied but nothing happened"
- Wait 10-15 seconds
- Check customer replied to the RIGHT SMS
- Check customer sent just the code (no extra text)
- Try again if needed

### "Can I enter the code somewhere?"
- No, customer must reply to the SMS
- This is how MTN/Vodafone/AirtelTigo security works
- We cannot bypass this step

## 🚨 Troubleshooting

### Payment Stuck on "Waiting for Approval"
**Cause**: Customer hasn't replied to SMS yet
**Fix**: Ask customer to check phone and reply with code

### Payment Shows "Failed"
**Possible causes**:
- Customer has insufficient balance
- Customer entered wrong code
- Code expired (10 minutes)
- Network issues

**Fix**: Try again with a new payment

### Customer Can't Find SMS
**Check**:
- Phone number entered correctly?
- Customer's phone has signal?
- SMS inbox not full?

**Fix**: Try again or use different payment method

## ✅ Success Indicators

You know payment is successful when:
- ✅ System shows "Payment Successful!" message
- ✅ Green checkmark appears
- ✅ Receipt can be printed
- ✅ Sale is completed

## 💡 Pro Tips

1. **Be Patient**: First-time customers may take longer
2. **Stay Calm**: If customer is confused, explain calmly
3. **Have Backup**: Keep cash/card option available
4. **Check Phone Number**: Verify before charging
5. **Explain Early**: Tell customer about SMS before starting
